## How to Run this Project

**IMPORTANT**:
Make sure that the following line in `src/SpotifyConfig.js` is uncommented:

```
export const redirectUri = "http://localhost:3000/";
```

Run the following commands to run the project:

```
npm install
```

```
npm start
```
